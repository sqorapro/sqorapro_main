# Lovable Prompt — Beauty Salon & Barbershop "NOIR"

## КРИТИЧЕСКИ ВАЖНО — ЧИТАЙ ПЕРВЫМ

**НЕ делай стандартный Lovable-дизайн.** Забудь про:
- синие/фиолетовые градиенты
- карточки в равномерном grid 3×N
- hero с текстом слева и картинкой справа
- иконки Lucide в кружочках с серым фоном
- стандартные shadcn Button/Card компоненты как основу

Этот сайт должен выглядеть как **лендинг люксового европейского салона**, который снял профессиональный фотограф и сверстал арт-директор модного журнала. Референсы по духу: Aesop, Byredo, Central Saint Martins.

## АНИМАЦИИ (Framer Motion)

```javascript
// Стандартный вариант появления для всех секций
const fadeUp = {
  initial: { opacity: 0, y: 40 },
  whileInView: { opacity: 1, y: 0 },
  viewport: { once: true, margin: "-80px" },
  transition: { duration: 0.8, ease: [0.25, 0.1, 0.25, 1] }
}

// Для заголовков H1/H2 — строка за строкой
const lineReveal = {
  initial: { opacity: 0, y: 60 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 1, ease: [0.25, 0.1, 0.25, 1], staggerChildren: 0.12 }
}

// Для строк в прайс-листе — появление одна за одной
staggerChildren: 0.06
```

**НЕ использовать:** `spring`, `bounce`, `rotate` анимации — они дешевеют визуал.
