# Lovable Prompt — Dental Clinic "Дента Люкс" / DentaLux

## ЗАДАЧА

Создай красивый, современный, полностью рабочий сайт-лендинг для частной стоматологической клиники **«Дента Люкс»** (Алматы, Казахстан). Сайт должен выглядеть как реально работающий бизнес — с живыми данными, конкретными ценами, фото-заглушками высокого качества, реальными именами врачей и настоящими отзывами.

---

## ДИЗАЙН-КОНЦЕПЦИЯ

**Стиль:** Чистый, дорогой, медицинский минимализм с тёплыми акцентами.  
**Палитра:**
- Основной фон: `#FFFFFF` (белый)
- Вторичный фон секций: `#F8F6F3` (тёплый офф-вайт, слоновая кость)
- Акцент: `#C8A96E` (тёплое золото — ассоциация с премиумом, не дешевизной)
- Текст основной: `#1A1A2E` (глубокий тёмно-синий, не чёрный)
- Текст вторичный: `#6B7280`
- Зелёный успех (индикатор "онлайн", галочки): `#22C55E`

**Шрифты:**
- Заголовки: `Playfair Display` (serif, элегантность)
- Тело: `Inter` (чистота, читаемость)

**Общее ощущение:** Когда пациент открывает сайт — он должен почувствовать, что попал в дорогую европейскую клинику, а не в районную поликлинику. Больше воздуха, меньше текста, больше доверия.

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

- **Фреймворк:** React + TypeScript
- **Стили:** Tailwind CSS
- **Компоненты:** shadcn/ui где уместно
- **Иконки:** Lucide React
- **Анимации:** Framer Motion — плавные появления секций при скролле (fadeInUp, staggerChildren)
- **Шрифты:** Google Fonts — Playfair Display + Inter
- **Форма:** только фронт (console.log при сабмите + toast «Спасибо! Перезвоним в течение 5 минут»)
- **Адаптив:** полностью мобильный (breakpoints: sm/md/lg/xl)
- **Табы в прайсе:** useState, без внешней библиотеки
- **Карусель отзывов:** автоплей каждые 5 сек, точки-индикаторы, свайп на мобильном
- **Sticky navbar:** прозрачный на hero, белый с тенью после скролла 80px
- **Scroll-to-section:** все ссылки навбара — плавный скролл

## ВАЖНЫЕ ДЕТАЛИ КОТОРЫЕ ОТЛИЧАЮТ ХОРОШИЙ САЙТ ОТ ПЛОХОГО

1. **Нет стоковых фраз** типа «Мы рады приветствовать вас». Только конкретика.
2. **Цены реальные** — не «уточняйте», а конкретные цифры в тенге.
3. **Врачи с именами** — не «наши специалисты», а живые люди с биографией.
4. **Отзывы с датами** — не «довольный клиент», а имя и месяц.
5. **Лицензия в футере** — сигнал доверия для медицинской ниши.
6. **Рассрочка Kaspi** — важнейший конверсионный элемент для КЗ.
7. **Netflix на потолке** — запоминающаяся деталь, реально используется в топ-клиниках.
8. **2GIS рейтинг** — местная платформа, важнее Google для Алматы.
